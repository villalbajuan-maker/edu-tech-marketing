# Aprendizajes

Este archivo captura aprendizajes confirmados, no solo ideas.

## Aprendizajes actuales

- El proyecto no parte de cero: ya existen plataforma, curso, contenidos, evaluaciones y presentacion comercial.
- El activo principal no es solamente el curso, sino la combinacion entre curso, plataforma propia, medicion y capacidad de adaptacion.
- El primer segmento de validacion debe ser colegios privados.
- Las empresas con responsabilidad social pueden ser un segundo segmento o canal de patrocinio, pero no deben distraer la primera validacion.
- Camilo es la fuente principal de verdad sobre historia, producto, pedagogia y capacidades reales.
- Juan Carlos debe estructurar el workbench como sistema operativo de colaboracion.
- Leonardo debe responsabilizarse por activacion comercial, prospectos y conversaciones.
- La primera oferta no deberia exigir comprar todo el curso de entrada; un diagnostico puede abrir mejor la conversacion.

## Aprendizajes por validar

- Si los colegios privados sienten urgencia real frente a educacion financiera.
- Si el argumento PISA/OCDE/MEN ayuda a abrir puertas o debe traducirse a beneficios mas directos.
- Quien decide dentro del colegio: rectoria, coordinacion academica, bienestar, area financiera, propietarios o consejo directivo.
- Si el primer producto vendible es diagnostico, piloto, licencia institucional o paquete completo.
- Que precio o modelo de cobro es aceptable para colegios privados.
- Que evidencia necesita un colegio para avanzar.
- Que objeciones aparecen con mas frecuencia.

## Como registrar aprendizajes

Cada nuevo aprendizaje debe incluir:

- fecha;
- fuente;
- que se aprendio;
- que cambia en la estrategia;
- si requiere decision o accion.
