# V1 - Diagnostico Escolar de Competencias Financieras

Esta carpeta contiene la version 1 congelada del Diagnostico Escolar de Competencias Financieras.

La V1 existe para probar la escalera comercial:

```text
Diagnostico -> Reporte vivo -> Piloto -> Implementacion institucional
```

## Principio

La V1 debe ser suficientemente clara para venderse, aplicarse, medir, entregar resultados y recomendar un piloto. No intenta ser la version final de plataforma, prueba adaptativa, benchmark nacional ni certificacion psicometrica.
