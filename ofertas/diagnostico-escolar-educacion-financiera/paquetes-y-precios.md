# Paquetes y precios

Este documento propone una estructura comercial inicial. Los precios deben definirse despues de revisar costos, alcance y posicionamiento.

## Paquete 1: Diagnostico exploratorio

### Para que sirve

Abrir una primera conversacion con bajo riesgo para el colegio.

### Incluye

- Aplicacion a un grupo limitado.
- Resultado general.
- Informe corto.
- Reunion de lectura.

### Mejor uso

Colegios que quieren ver valor antes de comprometerse.

## Paquete 2: Diagnostico institucional

### Para que sirve

Entregar una lectura mas seria del estado de educacion financiera del colegio.

### Incluye

- Aplicacion a varios grupos o grados.
- Resultados por dimension.
- Informe ejecutivo.
- Reunion con directivos.
- Recomendacion de implementacion.

### Mejor uso

Colegios con interes real en decidir una ruta.

## Paquete 3: Diagnostico + piloto

### Para que sirve

Medir, intervenir con una unidad del curso y volver a medir.

### Incluye

- Diagnostico inicial.
- Acceso acotado a plataforma.
- Unidad o modulo del curso.
- Evaluacion posterior.
- Reporte de avance.
- Propuesta de implementacion completa.

### Mejor uso

Colegios que quieren probar antes de comprar la licencia institucional.

## Preguntas para definir precio

- Cuantos estudiantes incluye cada paquete?
- Que costo tiene la aplicacion y analisis?
- Que tanto acompanamiento requiere?
- El diagnostico se cobrara o se usara como entrada subsidiada?
- Que precio se siente razonable para un colegio privado?
- Que precio posiciona la oferta como seria y no como una encuesta barata?
