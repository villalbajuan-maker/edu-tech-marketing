# Checklist de lanzamiento

## Producto

- [ ] Validar dimensiones con Camilo.
- [ ] Definir numero de preguntas.
- [ ] Confirmar si se usa la plataforma actual para aplicar el diagnostico.
- [ ] Definir escala de resultados.
- [ ] Preparar ejemplo de reporte.

## Comercial

- [ ] Definir nombre comercial final.
- [ ] Definir 3 paquetes iniciales.
- [ ] Definir si el primer diagnostico se cobra, se subsidia o se ofrece como piloto.
- [ ] Ajustar mensajes de acercamiento.
- [ ] Preparar guion de llamada.

## Operacion

- [ ] Definir responsable de aplicacion.
- [ ] Definir tiempos.
- [ ] Definir politica de datos.
- [ ] Definir formato de entrega.
- [ ] Definir siguiente paso despues del reporte.

## Validacion

- [ ] Seleccionar 10 colegios.
- [ ] Contactar 5.
- [ ] Conseguir 3 conversaciones.
- [ ] Presentar el diagnostico.
- [ ] Registrar objeciones.
- [ ] Ajustar oferta.
