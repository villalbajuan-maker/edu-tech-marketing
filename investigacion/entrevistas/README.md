# Entrevistas

Aqui se guardaran notas de entrevistas con colegios, docentes, directivos, empresas, fundaciones o aliados.

## Formato sugerido

Crear un archivo por entrevista:

```text
AAAA-MM-DD-nombre-institucion.md
```

Usar la plantilla de [validacion/plantilla-notas-de-llamada.md](../../validacion/plantilla-notas-de-llamada.md).
