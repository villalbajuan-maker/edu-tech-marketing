# Colaboracion

Este proyecto necesita combinar memoria fundacional, arquitectura estrategica y activacion comercial.

## Camilo

Camilo es el referente fundacional del proyecto.

Creo la plataforma, diseno su base pedagogica e instruccional, estructuro el curso, conoce la historia real del proyecto y ha liderado esfuerzos de implementacion y comercializacion.

Su responsabilidad dentro del workbench es validar que la estrategia, los documentos, las ofertas, los mensajes y las promesas comerciales correspondan con lo que el proyecto realmente es, ha sido y puede entregar.

Camilo debe revisar especialmente:

- historia del proyecto;
- capacidades reales de la plataforma;
- modelo pedagogico;
- diseno instruccional;
- contenido del curso;
- aprendizajes previos;
- promesas comerciales;
- limites actuales;
- oportunidades de evolucion.

## Juan Carlos

Juan Carlos lidera la arquitectura del workbench.

Su responsabilidad es organizar la informacion, estructurar la tesis de negocio, disenar el sistema de colaboracion, preparar la ruta de validacion y convertir el conocimiento del proyecto en una base clara para tomar decisiones, construir activos y escalar.

Juan Carlos debe cuidar que el repo funcione como fuente de verdad y no como acumulacion de documentos.

## Leonardo

Leonardo lidera la activacion comercial.

Su responsabilidad es identificar prospectos, conseguir contactos, abrir conversaciones, registrar oportunidades y alimentar el proceso de validacion con informacion real del mercado.

El primer foco de Leonardo son colegios privados. El segundo foco, cuando haya mas evidencia, seran empresas con responsabilidad social, fundaciones o aliados que puedan patrocinar implementaciones.

## Como se toman decisiones

- Las decisiones sobre historia, pedagogia, plataforma y capacidades reales requieren validacion de Camilo.
- Las decisiones sobre arquitectura del workbench, documentacion, roadmap y sistema de ejecucion las lidera Juan Carlos con validacion de Camilo.
- Las decisiones sobre prospeccion, contactos y activacion comercial las lidera Leonardo con alineacion de Camilo y Juan Carlos.
- Las decisiones estrategicas mayores se toman entre Camilo y Juan Carlos, usando evidencia comercial traida por Leonardo.

## Principio de colaboracion

El repo debe preservar aprendizaje, pero tambien empujar accion.

No se documenta para parecer organizados. Se documenta para vender mejor, decidir mejor, construir mejor y colaborar mejor.
