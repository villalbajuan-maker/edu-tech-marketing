# Modelo de negocio

## Modelos posibles

### Licencia institucional

El colegio paga por acceso al curso para estudiantes, docentes o grupos.

### Diagnostico

El colegio aplica un diagnostico inicial de competencias financieras y recibe un reporte.

### Piloto institucional

El colegio implementa una version acotada con diagnostico, modulo inicial, medicion y reporte.

### Paquete completo

Curso + plataforma + evaluaciones + seguimiento + reportes + acompanamiento.

### Patrocinio empresarial

Empresas o fundaciones financian la implementacion en colegios como programa de responsabilidad social.

## Modelo a validar primero

Diagnostico o piloto como entrada comercial hacia licencia o paquete institucional.

## Preguntas abiertas

- Cual es el rango de precio aceptable?
- Se cobra por estudiante, grupo, institucion o implementacion?
- El diagnostico debe ser gratuito, pagado o subsidiado?
- Que parte del valor percibe mas el colegio: contenidos, medicion, cumplimiento, diferenciacion o acompanamiento?
