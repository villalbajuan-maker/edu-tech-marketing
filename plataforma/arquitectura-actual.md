# Arquitectura actual

## Estado

La plataforma esta desarrollada con PHP y CodeIgniter.

## Pendiente de documentacion tecnica

Camilo debe ayudar a precisar:

- version de PHP;
- version de CodeIgniter;
- estructura general;
- modulos principales;
- base de datos;
- hosting actual;
- gestion de usuarios;
- gestion de cursos;
- reportes;
- evaluaciones;
- estado del codigo;
- si conviene crear un repositorio tecnico separado.

## Relacion con este workbench

Este repositorio documenta estrategia, negocio, validacion y direccion.

El codigo de plataforma deberia vivir en un repositorio tecnico separado cuando sea momento de trabajarlo formalmente.
