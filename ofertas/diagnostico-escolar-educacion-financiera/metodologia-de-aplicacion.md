# Metodologia de aplicacion

## Objetivo

Definir como se aplicaria el Diagnostico Escolar de Educacion Financiera en un colegio privado.

## Flujo propuesto

### 1. Conversacion inicial

Se valida si el colegio tiene interes, quien decide y que grupo podria participar.

Resultado esperado:

- colegio calificado;
- persona responsable identificada;
- grupo o grado tentativo;
- siguiente paso acordado.

### 2. Definicion del alcance

Se define si el diagnostico sera exploratorio, institucional o diagnostico + piloto.

Variables:

- numero de estudiantes;
- grados;
- tiempo disponible;
- modalidad;
- responsable interno;
- fecha de aplicacion;
- formato de reporte.

### 3. Preparacion

Se prepara el instrumento, acceso a plataforma o enlace de aplicacion, instrucciones y comunicacion interna.

### 4. Aplicacion

Los estudiantes responden el diagnostico en una sesion controlada.

Duracion sugerida:

- 25 a 35 minutos para version inicial.

### 5. Analisis

Se consolidan resultados por estudiante, grupo, grado o institucion segun alcance.

### 6. Reporte

Se entrega un informe ejecutivo con brechas, hallazgos y recomendaciones.

### 7. Reunion ejecutiva

Se presenta el resultado al colegio y se propone una ruta de implementacion.

## Roles

### Colegio

- Define grupo participante.
- Autoriza aplicacion.
- Coordina espacio y estudiantes.
- Recibe resultados.

### Finanzas Desde La Escuela

- Prepara instrumento.
- Aplica o acompana aplicacion.
- Analiza resultados.
- Entrega informe.
- Recomienda siguiente paso.

## Criterios de cuidado

- No prometer comparaciones nacionales si no hay base de datos suficiente.
- No presentar resultados individuales sensibles sin definir politica de manejo de datos.
- No convertir el diagnostico en examen punitivo.
- Comunicarlo como herramienta de mejora institucional.
