# Contexto normativo y educativo

## Referentes

El proyecto se apoya en referentes nacionales e internacionales de educacion financiera:

- OCDE.
- Pruebas PISA.
- Ministerio de Educacion Nacional.
- Orientaciones pedagogicas para educacion economica y financiera.
- Decreto 457 de 2014.
- Ley 1328 de 2009.
- Ley 1450 de 2011.

## Uso correcto del contexto

Este contexto no debe usarse como lista decorativa.

Debe servir para:

- justificar la relevancia del tema;
- mostrar alineacion educativa;
- abrir conversaciones institucionales;
- conectar la oferta con competencias medibles;
- evitar que el proyecto parezca un curso aislado.

## Cuidado comercial

El marco normativo puede abrir puertas, pero no necesariamente vende por si solo.

En conversaciones con colegios privados, debemos traducirlo a beneficios concretos:

- implementacion sencilla;
- valor agregado para estudiantes y familias;
- reportes;
- competencias para la vida;
- diferenciacion institucional;
- preparacion para retos reales.
