# Relacion con repositorio tecnico

Este workbench no debe convertirse en el repositorio principal del codigo de la plataforma.

## Funcion de este repo

- Tesis.
- Estrategia.
- Validacion.
- Producto.
- Comercial.
- Roadmap.
- Decisiones.

## Funcion del futuro repo tecnico

- Codigo de plataforma.
- Issues tecnicos.
- Releases.
- Documentacion de instalacion.
- Gestion de bugs.
- Integraciones.
- Desarrollo de tutor IA.

## Regla de conexion

Las decisiones tecnicas importantes deben estar conectadas con necesidades documentadas en este workbench.
