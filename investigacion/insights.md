# Insights

## Insights iniciales

- La plataforma propia es un activo estrategico porque permite adaptar la solucion y no depender de LMS externos.
- La educacion financiera debe venderse como formacion para la vida, no solo como cumplimiento.
- Los reportes pueden ser uno de los elementos mas valiosos para colegios.
- El diagnostico puede funcionar como puente entre interes y compra.
- La promesa de tutor IA es atractiva, pero no debe desplazar la validacion comercial inicial.

## Pendientes de investigacion
